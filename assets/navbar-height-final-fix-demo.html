🎯 Final Fix: Desktop Navbar Height!

Fixed the root cause: menu-lg class was adding extra padding (0.75rem top/bottom = 24px total). Reduced to 0.25rem for consistent height.

🔍 Root Cause Found

/* Documentation layout has: */
.menu-lg li > * {
  padding-top: 0.75rem;    /* 12px */
  padding-bottom: 0.75rem; /* 12px */
}
/* Total extra height: 24px! */

This made doc navbars 24px taller than custom navbars on desktop.

✅ Solution Applied

/* New override: */
.navbar .menu-lg li > * {
  padding-top: 0.25rem !important;    /* 4px */
  padding-bottom: 0.25rem !important; /* 4px */
}
/* Total extra height: 8px (much better!) */

Now both custom and doc navbars have consistent height.
